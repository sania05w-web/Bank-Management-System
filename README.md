# Bank Management System

## 📌 Overview
The Bank Management System is a console-based application developed using **Core Java**, **JDBC**, and **MySQL**. It provides basic banking functionalities such as customer registration, account management, deposits, withdrawals, and balance inquiries. The project demonstrates Java programming concepts and database connectivity using JDBC.

## 🚀 Technologies Used
- Core Java
- JDBC
- MySQL
- Eclipse IDE

## ✨ Features
- Customer Registration
- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- View Customer Details
- Update Customer Details
- Delete Customer Account

## 🛠️ Requirements
- Java JDK 8 or above
- Eclipse IDE
- MySQL Server
- MySQL Connector/J (JDBC Driver)

## ▶️ How to Run
1. Clone this repository.
2. Import the project into Eclipse IDE.
3. Create the required MySQL database and tables.
4. Add the MySQL Connector/J JAR to the project's build path.
5. Update the database username and password in the JDBC connection code.
6. Run the main Java class.
